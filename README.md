1 day 1 coding with TypeScript

## Requirement

* [npm](https://www.npmjs.com) (v9.8.1+)
* [Node.js](https://nodejs.org) (v18.18.0+)

## Initialize

```bash
$ npm install
```

## Build all

```bash
$ npm run build:main
```

## Test on Jest

```bash
$ npm run test
```

## Run on Node.js

```bash
$ npm run ts --file=./src/main/boj/no1000.ts
```

## Format on Prettier

```bash
$ npm run format
$ npm run format:fix
```

## Lint on ESLint

```bash
$ npm run lint
$ npm run lint:fix
```
